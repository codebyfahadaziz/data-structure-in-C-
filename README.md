# data-structure-in-C-
This repository contains beginner-friendly implementations of core Data Structures in C++.  All programs are written in easy C++ syntax, compatible with older compilers like Dev C++, and  topic-wise for easy learning, practice, and exam preparation.  The repository is ideal for students, DSA beginners, and anyone who wants to understand the dsa
